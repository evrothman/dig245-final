Combining Terms

Combining terms in your stylesheet can be useful for a handful of reasons. First, it can allow you to streamline your stylesheet by only needing to define a set of styles for multiple selectors at the same time. This is done by seperating the elements with a comma. For example, the styles to the left will apply to both the class and all paragraph elements.

Stylesheet Formatting 

      .className, p{
        color: black;
        text-align: left;
      }
Another way that you can combine terms is using "decendant" elements. In css, a decendant is a element that is nested inside another element. Essentally, you are able to style only the classes, ids, etc that are within another class, id, etc. For example, the code on the right will only style the class when that class is nested inside of a div. 

    div .className{
      color: black;
      text-align: left;
    }
The third way that you can combine terms is by only targeting selectors with a particular class. This can be helpful when a class is used frequently within your code, but you need to style some of these element slightly differently. Using the code on the right you can target only paragraphs with the class. Similar notation is used for psuedo classes, but these are more complex and require a section of their own. 

    p.className{
      color: red;
      text-align: right;
    }
