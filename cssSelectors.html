CSS Selectors

Selectors are elements that allows you to change the appearance of a specific piece of your html. There are several different types of selectors. The first of these we'll call html selectors, which are the tags used to build the page. Next are classes & ids, which the user can assign to parts of the html code. The third is interactive selectors, also called psuedo classes. These change according to how the user has interacted with your page. You can also use any combination of these to be even more specific in where to apply these changes.
