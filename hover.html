Hover

The most important thing to remember about the hover element is that is that it is appended to the end of an html element, class, or id using a colon. What this allows you to do is change the look of your element when a user is hovering over it just as you would any other css element. for example, if you hover over the selector above, the text color will change. While this is a common use case, you can change any styling based on whether or not a user is hovering.


Stylesheet Formatting 

        #yourID:hover{
          color:red;
        }
