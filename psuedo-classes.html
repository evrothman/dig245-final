Psuedo-Classes

Psuedo-classes allow your page's styling to change based on different states, most often user interaction. You'll see them in use any time you access a website, though you might not have know it. For example, when an element in a navigation bar changes color and gains an underline when you hover over it (as they do on this site), the person who build the website is using psuedo classes. On the following pages, you'll learn more about common use cases and how to implement psuedo-classes into your own websites.
