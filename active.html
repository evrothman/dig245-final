@media Rule

The @media rule is one of the more complicated elements of CSS, but is extremely useful in improving the usability of a website. Returning to the example of bootstrap, the column system it uses changes based on the width of a user's view window. This is something that we can recreate with the @media rule. Essentially, the @media rule brings in data from a user's computer, and then a coder (like youself) can tell the website to look a certain way based on whether that is the case. These data include the view width, view height, device orientation, and many others. After you have defined your media rule, you place all the style changes within the curly brackets of that media rule. For example, the box to the right will move below this text when the screen is small enough, and the color of the red box will change to grey.


stylesheet formatting

      @media (max-width: 780px){
       .className{
          background-color: #555;
          width: 90vw;
        }
      }
